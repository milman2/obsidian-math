# <span class="header-definition">Definition</span>

## Generator of a Group^[군의 생성원]

### Single Generator^[단일 생성원]

Group $G$와 원소 $g \in G$에 대해, **$g$에 의해 생성된 subgroup**^[부분군]:

$$\langle g \rangle = \{g^n : n \in \mathbb{Z}\}$$

**의미**: $g$를 반복적으로 곱하거나 역원을 취해 얻을 수 있는 모든 원소

**Additive notation** (abelian groups):
$$\langle g \rangle = \{ng : n \in \mathbb{Z}\}$$

### Properties of $\langle g \rangle$

1. $\langle g \rangle \leq G$ (subgroup)
2. $\langle g \rangle$는 $g$를 포함하는 **최소** subgroup
3. $|\langle g \rangle| = \text{ord}(g)$ (원소 $g$의 위수)

### Multiple Generators^[다중 생성원]

Subset $S \subseteq G$에 대해, **$S$에 의해 생성된 subgroup**:

$$\langle S \rangle = \bigcap_{\substack{H \leq G \\ S \subseteq H}} H$$

$S$를 포함하는 모든 subgroups의 교집합

**동등한 정의**:
$$\langle S \rangle = \{s_1^{e_1} s_2^{e_2} \cdots s_k^{e_k} : s_i \in S, e_i \in \mathbb{Z}, k \geq 0\}$$

$S$의 원소들과 그 역원들의 유한 곱

## Generating Set^[생성 집합]

Subset $S \subseteq G$가 $G$의 **generating set**^[생성 집합]이다 $\Leftrightarrow$

$$\langle S \rangle = G$$

**표기**: "$S$ generates $G$" 또는 "$G$ is generated by $S$"

### Minimal Generating Set^[최소 생성 집합]

Generating set $S$가 **minimal**^[최소]이다 $\Leftrightarrow$

$$\forall s \in S, \quad \langle S \setminus \{s\} \rangle \neq G$$

어떤 원소를 제거해도 더 이상 $G$를 생성하지 못함

## Cyclic Group^[순환군]

Group $G$가 **cyclic**^[순환]이다 $\Leftrightarrow$ 단일 원소로 생성됨

$$\exists g \in G, \quad G = \langle g \rangle$$

$g$를 $G$의 **generator**^[생성원]라 함

### Finite Cyclic Groups

$$G = \langle g \rangle \text{ with } |G| = n$$

**원소**: $G = \{e, g, g^2, \ldots, g^{n-1}\}$

**위수**: $\text{ord}(g) = n$

**표준 예**: $\mathbb{Z}/n\mathbb{Z} = \langle [1] \rangle$

### Infinite Cyclic Groups

$$G = \langle g \rangle \text{ with } |G| = \infty$$

**원소**: $G = \{\ldots, g^{-2}, g^{-1}, e, g, g^2, \ldots\}$

**위수**: $\text{ord}(g) = \infty$

**표준 예**: $\mathbb{Z} = \langle 1 \rangle$

## Finitely Generated Group^[유한 생성 군]

Group $G$가 **finitely generated**^[유한 생성]이다 $\Leftrightarrow$

$$\exists \text{ finite set } S, \quad G = \langle S \rangle$$

유한개의 원소로 생성 가능

**예**:
- 모든 finite groups (유한 생성)
- $\mathbb{Z}^n$ (유한 생성, rank $n$)
- $\mathbb{Q}$ (NOT finitely generated as group)

## Generators in Vector Spaces^[벡터 공간의 생성원]

Vector space $V$와 subset $S \subseteq V$에 대해:

### Span^[생성 공간]

$$\text{span}(S) = \left\{\sum_{i=1}^k c_i v_i : v_i \in S, c_i \in \mathbb{F}, k \geq 0\right\}$$

$S$의 원소들의 모든 linear combinations^[선형 결합]

**차이점**: 
- Group: 곱과 역원
- Vector space: 선형 결합

### Basis^[기저]

$S$가 **basis**^[기저]이다 $\Leftrightarrow$

1. $\text{span}(S) = V$ (generates)
2. $S$는 linearly independent^[선형 독립]

**의미**: Minimal generating set (vector space 관점)

자세한 내용은 [[Linear mapping]] 참조

## Generators in Other Structures

### Ring Generators^[환의 생성원]

Ring $R$과 subset $S$에 대해, **$S$에 의해 생성된 subring**:

$$\langle S \rangle_R = \text{smallest subring containing } S$$

$S$의 원소들의 합과 곱으로 만들 수 있는 모든 원소

### Ideal Generators^[아이디얼 생성원]

Ring $R$과 subset $S$에 대해, **$S$에 의해 생성된 ideal**:

$$(S) = \left\{\sum_{i=1}^k r_i s_i : r_i \in R, s_i \in S\right\}$$

**Principal ideal**^[주 아이디얼]: 단일 원소로 생성
$$(a) = Ra = \{ra : r \in R\}$$

자세한 내용은 [[Ring]] 참조

# <span class="header-examples">Examples</span>

## Example 1: $\mathbb{Z}$ (Integers)

$$\mathbb{Z} = \langle 1 \rangle = \langle -1 \rangle$$

**Generators**: $1$ 또는 $-1$

**Subgroups**: $n\mathbb{Z} = \langle n \rangle$ for $n \geq 0$

**특징**: Infinite cyclic group, 정확히 2개의 generators

## Example 2: $\mathbb{Z}/n\mathbb{Z}$ (Modular Integers)

$$\mathbb{Z}/n\mathbb{Z} = \langle [1] \rangle$$

**Generators**: $[a]$ where $\gcd(a, n) = 1$

**개수**: $\phi(n)$ (Euler's totient function^[오일러 파이 함수])

**예**: $\mathbb{Z}/12\mathbb{Z}$
- Generators: $[1], [5], [7], [11]$ (총 $\phi(12) = 4$개)
- Non-generators: $[2], [3], [4], [6], [8], [9], [10]$

## Example 3: Klein Four-Group

$$V_4 = \{e, a, b, c\} \text{ where } a^2 = b^2 = c^2 = e$$

**NOT cyclic**: 단일 원소로 생성 불가능

**Generating sets**:
- $\{a, b\}$: $V_4 = \langle a, b \rangle$ (minimal)
- $\{a, c\}$: $V_4 = \langle a, c \rangle$ (minimal)
- $\{a, b, c\}$: $V_4 = \langle a, b, c \rangle$ (not minimal)

## Example 4: Symmetric Group $S_3$

$$S_3 = \langle (12), (123) \rangle$$

**Generators**: 
- Two elements: $(12)$ (transposition), $(123)$ (3-cycle)
- All transpositions: $S_3 = \langle (12), (13), (23) \rangle$
- Two transpositions: $S_3 = \langle (12), (23) \rangle$

**일반화**: $S_n = \langle (12), (23), \ldots, (n-1,n) \rangle$ (adjacent transpositions)

## Example 5: Dihedral Group $D_n$

$$D_n = \langle r, s \mid r^n = s^2 = e, srs = r^{-1} \rangle$$

**Generators**: 
- $r$: rotation by $\displaystyle\frac{2\pi}{n}$
- $s$: reflection

**Relations**: 
- $r^n = e$ (order of rotation)
- $s^2 = e$ (order of reflection)
- $srs = r^{-1}$ (conjugation relation)

**예**: $D_4$ (square의 대칭)
- 8 elements
- 2 generators with specific relations

## Example 6: Free Group $F_2$

$$F_2 = \langle a, b \rangle$$

**No relations!** (completely free)

**원소**: $a$, $b$, $a^{-1}$, $b^{-1}$, $ab$, $ba$, $a^2b^{-1}$, ...

**특징**: 
- Infinite group
- 2개 generators
- No non-trivial relations

## Example 7: $\mathbb{Z}^2$ (Free Abelian Group)

$$\mathbb{Z}^2 = \langle (1,0), (0,1) \rangle$$

**Standard generators**: $(1,0)$, $(0,1)$

**Other generating sets**:
- $\{(1,0), (1,1)\}$ - $\{(2,0), (0,2)\}$ ✗ (generates $2\mathbb{Z}^2 \subsetneq \mathbb{Z}^2$)
- $\{(1,1), (1,-1)\}$ 
**Rank**: 2 (최소 generators 개수)

## Example 8: $\mathbb{R}$ over $\mathbb{Q}$

$\mathbb{R}$를 $\mathbb{Q}$-vector space로 볼 때:

**NOT finitely generated!**

**이유**: $\dim_\mathbb{Q}(\mathbb{R}) = 2^{\aleph_0}$ (uncountable)

Hamel basis^[하멜 기저] 존재하지만 uncountable

## Example 9: Polynomial Ring $\mathbb{Q}[x]$

$$\mathbb{Q}[x] = \langle 1, x, x^2, x^3, \ldots \rangle_{\mathbb{Q}\text{-vector space}}$$

**As $\mathbb{Q}$-vector space**: Infinite basis $\{1, x, x^2, \ldots\}$

**As ring**: Generated by $\{x\}$ over $\mathbb{Q}$

## Example 10: Cyclic Subgroups in $\mathbb{Z}/12\mathbb{Z}$

| Generator | Subgroup | Order |
|-----------|----------|-------|
| $[1]$ | $\mathbb{Z}/12\mathbb{Z}$ | 12 |
| $[2]$ | $\{[0], [2], [4], [6], [8], [10]\}$ | 6 |
| $[3]$ | $\{[0], [3], [6], [9]\}$ | 4 |
| $[4]$ | $\{[0], [4], [8]\}$ | 3 |
| $[6]$ | $\{[0], [6]\}$ | 2 |

**관찰**: $|\langle [a] \rangle| = \displaystyle\frac{12}{\gcd(a, 12)}$

# <span class="header-properties">Properties</span>

## Generators of Cyclic Groups^[순환군의 생성원]

### Finite Cyclic Group

$G = \langle g \rangle$ with $|G| = n$에 대해:

**정리**: $g^k$가 generator $\Leftrightarrow$ $\gcd(k, n) = 1$

**Generators 개수**: $\phi(n)$ (Euler's totient function)

**증명**: $\langle g^k \rangle = G \Leftrightarrow \text{ord}(g^k) = n \Leftrightarrow \gcd(k,n) = 1$

### Infinite Cyclic Group

$G = \langle g \rangle$ with $|G| = \infty$에 대해:

**Generators**: 정확히 $g$와 $g^{-1}$

## Generating Sets and Homomorphisms^[생성 집합과 준동형사상]

### Universal Property

$G = \langle S \rangle$이고 $\phi: S \to H$ (함수)가 주어졌을 때:

Homomorphism $\Phi: G \to H$ 존재 with $\Phi|_S = \phi$ $\Leftrightarrow$ $\phi$가 적절한 relations를 보존

**의미**: Generator들의 이미지만 정하면 homomorphism이 결정됨

### Extension of Homomorphisms

$G = \langle S \rangle$일 때:

$$\phi: G \to H \text{ homomorphism} \Rightarrow \phi \text{ determined by } \phi(S)$$

**결과**: 
- Homomorphisms는 generators의 이미지로 완전히 결정
- $\phi(g) = \phi(s_1^{e_1} \cdots s_k^{e_k}) = \phi(s_1)^{e_1} \cdots \phi(s_k)^{e_k}$

## Subgroups and Generators^[부분군과 생성원]

### Subgroups of Cyclic Groups

**정리**: Cyclic group의 모든 subgroup은 cyclic

**증명**: $G = \langle g \rangle$이고 $H \leq G$이면, $H = \langle g^d \rangle$ for some $d \mid n$

### Subgroup Lattice

$\mathbb{Z}/n\mathbb{Z}$의 subgroups:

$$\langle [d] \rangle \text{ for each divisor } d \mid n$$

**예**: $\mathbb{Z}/12\mathbb{Z}$
```
        Z/12Z = ⟨[1]⟩
       /   |   \
      /    |    \
  ⟨[2]⟩  ⟨[3]⟩  ⟨[4]⟩
      \    |    /
       \   |   /
        ⟨[6]⟩
          |
        {[0]}
```

## Rank^[계수]

Finitely generated abelian group $G$의 **rank**^[계수]:

$$\text{rank}(G) = \text{최소 generators 개수 (in free part)}$$

더 정확히: $G \cong \mathbb{Z}^r \oplus T(G)$일 때, $\text{rank}(G) = r$

**예**:
- $\text{rank}(\mathbb{Z}^n) = n$
- $\text{rank}(\mathbb{Z}/n\mathbb{Z}) = 0$ (finite, pure torsion)
- $\text{rank}(\mathbb{Z} \times \mathbb{Z}/5\mathbb{Z}) = 1$

자세한 내용은 [[Group]] 참조

## Generators and Direct Sums^[생성원과 직접합]

### Direct Sum of Cyclic Groups

$$G = \langle g_1 \rangle \oplus \langle g_2 \rangle \oplus \cdots \oplus \langle g_n \rangle$$

**Generators**: $G = \langle g_1, g_2, \ldots, g_n \rangle$ (필요 충분)

**독립성**: $g_i$들이 "independent"하게 작동

### Example

$$\mathbb{Z}/4\mathbb{Z} \times \mathbb{Z}/6\mathbb{Z} = \langle ([1], [0]), ([0], [1]) \rangle$$

**Order**: $|G| = 24$

자세한 내용은 [[Direct Sum]] 참조

# <span class="header-theorem">Theorem</span>

## Classification of Cyclic Groups^[순환군의 분류]

### Finite Cyclic Groups

모든 order $n$인 cyclic group은 isomorphic:

$$G \cong \mathbb{Z}/n\mathbb{Z}$$

**유일성**: Isomorphism을 제외하고 유일

### Infinite Cyclic Groups

모든 infinite cyclic group은 isomorphic:

$$G \cong \mathbb{Z}$$

**유일성**: Isomorphism을 제외하고 유일

자세한 내용은 [[Isomorphism]] 참조

## Fundamental Theorem of Finitely Generated Abelian Groups

Finitely generated abelian group $G$는 다음과 같이 분해:

$$G \cong \mathbb{Z}^r \times C_{p_1^{a_1}} \times C_{p_2^{a_2}} \times \cdots \times C_{p_k^{a_k}}$$

**Generators**:
- Free part: $r$개 generators (rank)
- Torsion part: Cyclic groups의 direct product

**Minimal generating set**: $r + k$개 원소

## Structure of $(\mathbb{Z}/n\mathbb{Z})^\times$

Multiplicative group of units:

$$(\mathbb{Z}/n\mathbb{Z})^\times = \{[a] : \gcd(a, n) = 1\}$$

### When is it Cyclic?

$$(\mathbb{Z}/n\mathbb{Z})^\times \text{ is cyclic} \Leftrightarrow n \in \{1, 2, 4, p^k, 2p^k\}$$

where $p$ is odd prime, $k \geq 1$

**Generator**: **Primitive root modulo $n$**^[원시근]

**예**:
- $(\mathbb{Z}/7\mathbb{Z})^\times = \langle [3] \rangle$ (cyclic, order 6)
- $(\mathbb{Z}/8\mathbb{Z})^\times = \{[1], [3], [5], [7]\}$ (NOT cyclic)

## Nielsen-Schreier Theorem

**정리**: Free group의 subgroup은 free

$F_n$이 rank $n$ free group이고 $H \leq F_n$이면, $H$는 free group

**결과**: Subgroup도 적절한 generating set 존재

# <span class="header-remark">Remark</span>

## 직관적 이해

**Generator**: "Building blocks" of a structure

**메타포**:
- **LEGO**: 기본 블록들로 모든 것을 만듦
- **DNA**: 4개 염기로 모든 유전 정보 생성
- **Alphabet**: 26개 문자로 모든 단어 생성

**핵심**: 
- 최소한의 정보로 전체 구조 복원
- Generators만 알면 전체 구조 이해 가능

## Generating Set vs Basis

| | **Generating Set** | **Basis** |
|---|---|---|
| 구조 | Group | Vector space |
| 연산 | 곱, 역원 | 선형 결합 |
| 유일성 | 표현 not unique | 표현 unique |
| 독립성 | Not required | Required (linear independence) |

**예**:
- $\mathbb{Z}^2 = \langle (1,0), (0,1), (1,1) \rangle$ (not minimal)
- $\mathbb{R}^2 = \text{span}\{(1,0), (0,1)\}$ (basis, minimal)

## Cyclic vs Non-cyclic

### Cyclic Groups

**특징**:
- 단일 generator
- Abelian (항상)
- Simple structure
- Well-understood

**예**: $\mathbb{Z}$, $\mathbb{Z}/n\mathbb{Z}$, subgroups of $(\mathbb{R}, +)$

### Non-cyclic Groups

**특징**:
- 다중 generators 필요
- 더 복잡한 구조
- Relations 중요

**예**: $V_4$, $S_n$ ($n \geq 3$), $D_n$ ($n \geq 3$)

## Presentation of Groups^[군의 표시]

**Group presentation**^[군 표시]:

$$G = \langle S \mid R \rangle$$

- $S$: Generators (생성원 집합)
- $R$: Relations (관계식 집합)

**예**: $D_4 = \langle r, s \mid r^4 = s^2 = e, srs = r^{-1} \rangle$

**의미**: $S$로 생성하되, $R$의 관계식을 만족

## 표기법

| 표기 | 의미 |
|------|------|
| $\langle g \rangle$ | $g$에 의해 생성된 subgroup |
| $\langle S \rangle$ | $S$에 의해 생성된 subgroup |
| $G = \langle g \rangle$ | $G$는 cyclic, generator $g$ |
| $\langle g \mid g^n = e \rangle$ | Presentation with relation |
| $\text{span}(S)$ | Linear span (vector spaces) |
| $(a)$ | Principal ideal generated by $a$ |

## Minimal vs Non-minimal

**Minimal generating set**:
- 어떤 원소도 제거 불가
- "Efficient" representation
- Not unique (일반적으로)

**Non-minimal generating set**:
- Redundant elements
- "Over-specified"
- 더 많은 flexibility

**예**: $\mathbb{Z}^2$
- Minimal: $\{(1,0), (0,1)\}$ (2개)
- Non-minimal: $\{(1,0), (0,1), (1,1)\}$ (3개)

## Common Pitfall^[흔한 실수]

### 1. Cyclic = Finite?

✗ Cyclic group이 항상 finite은 아님!

 $\mathbb{Z}$는 infinite cyclic group

### 2. Abelian = Cyclic?

✗ Abelian이라고 해서 cyclic은 아님

**반례**: $V_4 = C_2 \times C_2$ (abelian but not cyclic)

### 3. Generators 개수 = Minimal?

✗ Generating set이 minimal이 아닐 수 있음

**예**: $\mathbb{Z} = \langle 2, 3 \rangle$ (2개지만 $\langle 1 \rangle$로 충분)

### 4. $\langle S \rangle$ 계산

✗ $\langle a, b \rangle = \{a^m b^n : m,n \in \mathbb{Z}\}$?

✓ Non-abelian group에서는 더 복잡!
$$\langle a, b \rangle = \{a^{e_1}b^{e_2}a^{e_3}b^{e_4} \cdots : e_i \in \mathbb{Z}\}$$

### 5. Generator는 유일?

✗ 일반적으로 multiple generators 존재

 Finite cyclic group: $\phi(n)$개 generators
