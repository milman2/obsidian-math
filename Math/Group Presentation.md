# <span class="header-definition">Definition</span>

## Group Presentation^[군 표현]

**Group presentation**^[군 표현]:

$$\langle S \mid R \rangle$$

where:
- $S$ = set of **generators**^[생성원]
- $R$ = set of **relations**^[관계식] (equations)

**의미**: Group generated by $S$ subject to relations $R$

### Formal Definition

$$G = \langle S \mid R \rangle = F(S) / N$$

where:
- $F(S)$ = free group on $S$
- $N$ = normal closure of $R$ in $F(S)$

**Normal closure**: Smallest normal subgroup containing all elements in $R$

자세한 내용은 [[Free Group]], [[Quotient Group, Factor Group]] 참조

## Relations

**Relation**: Equation between words

$$r = e \quad \text{or} \quad w_1 = w_2$$

**예**:
- $a^3 = e$ (order 3)
- $ab = ba$ (commutativity)
- $aba^{-1}b^{-1} = e$ (commutator = identity)

**Convention**: Usually write $r$ instead of $r = e$

## Finite Presentation

**Finitely presented group**^[유한 표현 가능 군]:

$$G = \langle S \mid R \rangle$$

with both $S$ and $R$ **finite**

**의미**: Described by finite data

**Not all finitely generated groups** are finitely presented!

자세한 내용은 [[Finitely Generated Groups]] 참조

# <span class="header-properties">Properties</span>

## Every Group has Presentation

**정리**: Every group $G$ has presentation

$$G \cong \langle S \mid R \rangle$$

**Proof**: 
- Take $S = G$ (generators)
- $R$ = all relations in $G$
- Then $G = \langle S \mid R \rangle$ ✓

**문제**: This presentation might not be efficient!

## Non-uniqueness

**중요**: Presentation is **NOT unique**!

**예**: $\mathbb{Z}/6\mathbb{Z}$

$$\langle a \mid a^6 \rangle \cong \langle b \mid b^6 \rangle \cong \langle x, y \mid x^2, y^3, xy=yx \rangle$$

**Word problem**: Deciding if two presentations give same group is **undecidable**!

## Tietze Transformations

**Operations** preserving isomorphism class:

**T1**: Add/remove redundant generator

$$\langle a, b \mid R \rangle \leftrightarrow \langle a, b, c \mid R, c = w(a,b) \rangle$$

**T2**: Add/remove redundant relation

**T3**: Replace generator by word

**응용**: Show two presentations are equivalent

## Relations as Normal Subgroup

$$\langle S \mid R \rangle = F(S) / \langle\langle R \rangle\rangle$$

where $\langle\langle R \rangle\rangle$ = normal closure of $R$

**Normal closure**:

$$\langle\langle R \rangle\rangle = \{g r_i^{\epsilon_i} g^{-1} : g \in F(S), r_i \in R, \epsilon_i = \pm 1\}$$

**의미**: All conjugates of relations

## Abelianization

**Add commutator relations**:

$$G^{\text{ab}} = G / [G,G] = \langle S \mid R, [s_i, s_j] \text{ for all } i,j \rangle$$

**예**: $\langle a, b \mid \emptyset \rangle^{\text{ab}} = \langle a, b \mid ab=ba \rangle \cong \mathbb{Z}^2$

자세한 내용은 [[Commutator Subgroup]] 참조

## Word Problem

**Word problem**: Given $w$ in $\langle S \mid R \rangle$, is $w = e$?

**General case**: UNDECIDABLE! (Novikov-Boone)

**Special cases**: Decidable for some classes
- Free groups ✓
- Finite groups ✓
- Abelian groups ✓
- Hyperbolic groups ✓

# <span class="header-examples">Examples</span>

## Example 1: Cyclic Groups

$$\mathbb{Z} = \langle a \mid \emptyset \rangle$$

No relations! (infinite cyclic)

$$\mathbb{Z}/n\mathbb{Z} = \langle a \mid a^n \rangle$$

One relation: order $n$

자세한 내용은 [[Cyclic Group]] 참조

## Example 2: Dihedral Group

$$D_n = \langle r, s \mid r^n, s^2, (rs)^2 \rangle$$

where:
- $r$ = rotation by $2\pi/n$
- $s$ = reflection
- $r^n = e$, $s^2 = e$, $srs = r^{-1}$

자세한 내용은 [[Dihedral Group]] 참조

## Example 3: Free Group

$$F_n = \langle a_1, \ldots, a_n \mid \emptyset \rangle$$

No relations! Completely free

자세한 내용은 [[Free Group]] 참조

## Example 4: Free Abelian Group

$$\mathbb{Z}^n = \langle a_1, \ldots, a_n \mid [a_i, a_j] \text{ for all } i,j \rangle$$

Relations: All commutators = identity

자세한 내용은 [[Abelian Group]] 참조

## Example 5: Quaternion Group

$$Q_8 = \langle i, j \mid i^4, i^2 j^2, iji^{-1}j \rangle$$

Alternative:

$$Q_8 = \langle i, j, k \mid i^2 = j^2 = k^2 = ijk = -1 \rangle$$

## Example 6: Symmetric Group

$$S_n = \langle (12), (123\cdots n) \mid \text{relations} \rangle$$

Two generators suffice!

**Relations**: More complex

자세한 내용은 [[Symmetric Group]] 참조

## Example 7: Klein Four-Group

$$V_4 = \langle a, b \mid a^2, b^2, ab=ba \rangle$$

$$\cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}$$

## Example 8: Baumslag-Solitar Groups

$$BS(m,n) = \langle a, b \mid ba^m b^{-1} = a^n \rangle$$

**Special cases**:
- $BS(1,1) \cong \mathbb{Z}^2$
- $BS(1,n)$ metabelian
- $BS(2,3)$ not Hopfian

# <span class="header-theorem">Theorem</span>

## Von Dyck's Theorem

**정리**: If $\langle S \mid R \rangle$ and $\langle S \mid R' \rangle$ with $R \subseteq R'$,

$$\exists \text{ surjective homomorphism } \langle S \mid R \rangle \twoheadrightarrow \langle S \mid R' \rangle$$

**의미**: More relations → smaller (quotient) group

**예**: $F_2 = \langle a,b \mid \emptyset \rangle \twoheadrightarrow \mathbb{Z}^2 = \langle a,b \mid ab=ba \rangle$

## Existence and Uniqueness

**정리**: For any $S$ and $R$, $\exists ! G$ (up to isomorphism) with presentation $\langle S \mid R \rangle$

**Proof**: $G = F(S) / \langle\langle R \rangle\rangle$ ✓

## Tietze's Theorem

**정리**: Two presentations give isomorphic groups $\Leftrightarrow$ 

Can transform one to other via Tietze transformations

**문제**: No algorithm to decide this in general!

## Word Problem is Undecidable

**Novikov-Boone Theorem**: $\exists$ finitely presented group with undecidable word problem

**의미**: No algorithm to decide if $w = e$ for all $w$

**예**: $\langle a_1, \ldots, a_n \mid r_1, \ldots, r_m \rangle$ (specific construction)

## Higman Embedding Theorem

**정리**: Finitely generated group has decidable word problem $\Leftrightarrow$

Can be embedded in finitely presented group

**의미**: Finitely presented groups can be "wild"

## Dehn's Problems

Three fundamental problems (1911):

**Word problem**: Is $w = e$? (Undecidable)

**Conjugacy problem**: Is $w_1 \sim w_2$? (Undecidable)

**Isomorphism problem**: Is $G_1 \cong G_2$? (Undecidable)

# <span class="header-remark">Remark</span>

## 직관적 이해

**Presentation = "Recipe for building group"**

### 메타포: 레고 설명서

**Generators**: 블록 종류

**Relations**: 조립 규칙

**Group**: 완성된 구조

**예**: $\langle r, s \mid r^4, s^2, (rs)^2 \rangle$ = 정사각형의 대칭

### 메타포: 프로그래밍

**Generators**: 변수 선언

**Relations**: 제약 조건

**Group**: 만족하는 모든 값들

## 왜 중요한가?

### 1. Concrete Description

**Explicit**: 유한 개의 데이터로 group 표현

**Practical**: 계산 가능 (일부 경우)

### 2. Classification

**Structure**: 어떤 group인지 이해

**예**: $\langle a \mid a^{12} \rangle = \mathbb{Z}/12\mathbb{Z}$

### 3. Topology

**Fundamental groups**: $\pi_1(X)$의 presentation

**Covering spaces**: Presentations 통해 계산

### 4. Combinatorics

**Cayley graphs**: Presentation에서 그래프 구성

**Word problem**: Algorithmic questions

### 5. Geometric Group Theory

**Hyperbolic groups**: Presentations with special properties

**Small cancellation theory**: Geometric conditions on relations

## Generators vs Relations

| **More Generators** | **More Relations** |
|---|---|
| Easier to express elements | Smaller group |
| Fewer relations needed | Harder to verify $w \neq e$ |
| Larger free group | More constraints |

**Trade-off**: Efficiency vs simplicity

## Presentation 작성 방법

### Step 1: Generators 선택

**원칙**: Minimal but sufficient

**예**: $D_4$ → $\langle r, s \rangle$ (rotation, reflection)

### Step 2: Relations 찾기

**Orders**: $r^n = e$, $s^m = e$

**Interactions**: $srs^{-1} = r^{-1}$

**Dependencies**: 다른 relations에서 유도 가능한 것 제외

### Step 3: Verify

**모든 원소 표현 가능**?

**Relations 충분**?

**Independent**?

## 계산 방법

### Normal Form

**Goal**: 각 원소를 standard form으로 표현

**예**: $D_n$ → $r^i s^j$ where $0 \leq i < n$, $j \in \{0,1\}$

### Rewriting

**Relations**: Rewrite rules

**예**: $a^3 = e$ → $a^3 \to 1$

**Apply**: Repeatedly until irreducible

### Coset Enumeration

**Todd-Coxeter algorithm**: Compute subgroup index

**응용**: Find normal forms, check properties

## 표기법

| 표기 | 의미 |
|------|------|
| $\langle S \mid R \rangle$ | Presentation |
| $\langle a, b \mid r_1, r_2 \rangle$ | Explicit generators/relations |
| $[a,b]$ | Commutator $aba^{-1}b^{-1}$ |
| $\langle\langle R \rangle\rangle$ | Normal closure of $R$ |
| $F(S)$ | Free group on $S$ |

## Common Pitfall^[흔한 실수]

### 1. Non-uniqueness

Same group, many presentations!

**예**: $\langle a \mid a^6 \rangle \cong \langle x, y \mid x^2, y^3, xy=yx \rangle$

### 2. Redundant Relations

Some relations implied by others

**예**: $\langle a \mid a^6, a^3 \rangle = \langle a \mid a^3 \rangle$ (since $a^3 \mid a^6$)

### 3. Word Problem

Cannot always decide if $w = e$!

Undecidable in general

### 4. Finite Presentation ≠ Finite Group

$\langle a \mid \emptyset \rangle = \mathbb{Z}$ infinite but finitely presented!

### 5. Generators ≠ Free Generators

Generators have relations!

Only free if $R = \emptyset$

## 실용적 팁

**Tietze transformations**: Equivalent presentations

### Verification

**Check**: All relations independent?

**Compute**: Order, structure
