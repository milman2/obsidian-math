# <span class="header-definition">Definition</span>

## Commutator^[교환자]

원소 $a, b \in G$의 **commutator**^[교환자]:

$$[a, b] = aba^{-1}b^{-1}$$

**Alternative notation**: $[a, b] = a^{-1}b^{-1}ab$ (일부 문헌)

### 의미

**$[a,b] = e$ $\Leftrightarrow$ $ab = ba$** (교환)

$$[a,b] = aba^{-1}b^{-1} = e \Leftrightarrow aba^{-1} = b \Leftrightarrow ab = ba$$

**측정**: "$a$와 $b$가 얼마나 교환하지 않는가"

## Properties of Single Commutator

### Identity Cases

$$[a, b] = e \Leftrightarrow ab = ba$$

$$[a, e] = [e, b] = e$$

$$[a, a] = e$$

### Inverse

$$[a, b]^{-1} = (aba^{-1}b^{-1})^{-1} = bab^{-1}a^{-1} = [b, a]$$

**의미**: $[a,b]^{-1} = [b,a]$ (순서 바꾸면 inverse)

### Conjugation

$$g[a,b]g^{-1} = [gag^{-1}, gbg^{-1}]$$

**증명**: 
$$g(aba^{-1}b^{-1})g^{-1} = (gag^{-1})(gbg^{-1})(ga^{-1}g^{-1})(gb^{-1}g^{-1})$$
$$= (gag^{-1})(gbg^{-1})(gag^{-1})^{-1}(gbg^{-1})^{-1}$$

# <span class="header-examples">Examples</span>

## Example 1: Abelian Groups

$G$ abelian $\Rightarrow [a, b] = e$ for all $a, b \in G$

$$ab = ba \Rightarrow aba^{-1}b^{-1} = e$$

자세한 내용은 [[Abelian Group]] 참조

## Example 2: $S_3$

$$[(12), (13)] = (12)(13)(12)^{-1}(13)^{-1}$$
$$= (12)(13)(12)(13)$$
$$= (132)(13) = (123)$$

**확인**: $(12)(13) = (132) \neq (123) = (13)(12)$

## Example 3: Dihedral Group $D_n$

$$D_n = \langle r, s : r^n = s^2 = e, srs = r^{-1} \rangle$$

$$[r, s] = rsr^{-1}s^{-1} = rs(sr) = r(sr)s$$

Using $sr = r^{-1}s$:

$$[r, s] = r(r^{-1}s)s = rs^2 = r$$

Wait, let me recalculate:

$$[r,s] = rsr^{-1}s^{-1}$$

$srs = r^{-1}$ $\Rightarrow$ $sr = r^{-1}s$

$$[r,s] = rsr^{-1}s = r(sr)r^{-1}s = r(r^{-1}s)r^{-1}s = ssr^{-1}s = r^{-1}s^2 = r^{-1}$$

Actually: $s^2 = e$ so

$$[r,s] = rsr^{-1}s = rs(r^{-1}s) = rs \cdot sr^{-1} = r(sr)sr^{-1}$$

Using $sr = r^{-1}s$:

$$= rr^{-1}s^2r^{-1} = er^{-1} = r^{-1}$$ (if $n > 2$)

자세한 내용은 [[Dihedral Group]] 참조

## Example 4: Matrix Groups

$$A = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}, \quad B = \begin{pmatrix} 1 & 0 \\ 1 & 1 \end{pmatrix}$$

$$[A, B] = ABA^{-1}B^{-1}$$

**계산**: (2×2 matrices)

$$[A,B] = \begin{pmatrix} 1 & -1 \\ 1 & 0 \end{pmatrix}$$

(계산 생략)

## Example 5: Heisenberg Group

$$A = \begin{pmatrix} 1 & 1 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}, \quad B = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 1 \\ 0 & 0 & 1 \end{pmatrix}$$

$$[A, B] = \begin{pmatrix} 1 & 0 & 1 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}$$

**In center!**: Commutator is central

# <span class="header-properties">Properties</span>

## Hall-Witt Identity

$$[a, b^{-1}, c]^b [b, c^{-1}, a]^c [c, a^{-1}, b]^a = e$$

where $[x, y, z] = [[x, y], z]$ and $x^y = yxy^{-1}$

## Three Subgroups Lemma

$$[[A, B], C] = \{e\} \text{ and } [[B, C], A] = \{e\}$$

$$\Rightarrow [[C, A], B] = \{e\}$$

where $[A, B] = \langle [a, b] : a \in A, b \in B \rangle$

**응용**: Lower central series, Nilpotent groups

자세한 내용은 [[Nilpotent Group]] 참조

## Commutator Identities

### Basic

1. $[a, b] = e \Leftrightarrow ab = ba$
2. $[a, b]^{-1} = [b, a]$
3. $[a, bc] = [a, c][a, b]^c$ (where $x^y = yxy^{-1}$)
4. $[ab, c] = [a, c]^b[b, c]$

### Conjugation

$$g[a, b]g^{-1} = [gag^{-1}, gbg^{-1}]$$

# <span class="header-remark">Remark</span>

## 직관적 이해

**Commutator = "비가환성의 측정"**

### 메타포: 오차

**$[a,b]$**: "$ab$와 $ba$의 차이"

**$[a,b] = e$**: 완벽히 교환 (오차 없음)

**$[a,b] \neq e$**: 교환 안 함 (오차 존재)

### 메타포: 경로 차이

**$ab$**: $a$ 먼저, $b$ 나중

**$ba$**: $b$ 먼저, $a$ 나중

**$[a,b]$**: 두 경로의 "차이"

## 왜 중요한가?

### 1. Commutator Subgroup

$[G, G]$ generates commutator subgroup

자세한 내용은 [[Commutator Subgroup]] 참조

### 2. Nilpotent Groups

Lower central series: $G_i = [G, G_{i-1}]$

자세한 내용은 [[Nilpotent Group]] 참조

### 3. Lie Algebras

Commutator $\leftrightarrow$ Lie bracket

### 4. Quantum Mechanics

$[A, B] = AB - BA$ (commutator in physics)

## 계산 팁

**Step 1**: $ab$ 계산

**Step 2**: $ba$ 계산

**Step 3**: $aba^{-1} = ?$

**Step 4**: $aba^{-1}b^{-1} = ?$

**또는**: Conjugation relations 사용

## 표기법

| 표기 | 의미 |
|------|------|
| $[a, b]$ | $aba^{-1}b^{-1}$ |
| $[a, b, c]$ | $[[a, b], c]$ |
| $x^y$ | $yxy^{-1}$ (conjugation) |

## Common Pitfall^[흔한 실수]

### 1. 순서

$[a,b] \neq [b,a]$ (일반적으로)

$[a,b]^{-1} = [b,a]$

### 2. Notation

Some books: $[a,b] = a^{-1}b^{-1}ab$

Always check convention!

### 3. Product rules

$[ab, c] \neq [a,c][b,c]$ (일반적으로)

$[ab, c] = [a,c]^b[b,c]$ (correct)

## 관련 개념

- [[Abelian Group]]: All commutators trivial
- [[Nilpotent Group]]: Iterated commutators
