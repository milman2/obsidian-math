# <span class="header-definition">Definition</span>

## Lagrange's Theorem^[라그랑주 정리]

**정리**: Finite group $G$와 subgroup $H \leq G$에 대해

$$|H| \mid |G|$$

**즉**: Subgroup의 order는 group의 order를 나눈다

### Index^[지표]

$$[G : H] = \frac{|G|}{|H|}$$

**의미**: Left cosets의 개수 = Right cosets의 개수

**Lagrange's theorem**: $|G| = |H| \cdot [G : H]$

## Coset^[잉여류]

Subgroup $H \leq G$, 원소 $g \in G$에 대해:

### Left Coset^[좌잉여류]

$$gH = \{gh : h \in H\}$$

### Right Coset^[우잉여류]

$$Hg = \{hg : h \in H\}$$

### 성질

1. **크기**: $|gH| = |H| = |Hg|$
2. **Partition**: Left cosets는 $G$를 분할
3. **같거나 disjoint**: $g_1H = g_2H$ or $g_1H \cap g_2H = \emptyset$

---

# <span class="header-proof">Proof</span>

## Lagrange's Theorem 증명

### Step 1: Cosets의 크기

**주장**: 모든 left coset $gH$는 $|H|$개의 원소를 가짐

**증명**: 
$$\phi: H \to gH, \quad h \mapsto gh$$

- **Injective**: $gh_1 = gh_2 \Rightarrow h_1 = h_2$ ✓
- **Surjective**: By definition ✓

따라서 $|gH| = |H|$ ✓

### Step 2: Cosets는 Partition

**Equivalence relation**: $g_1 \sim g_2 \Leftrightarrow g_1H = g_2H$

**확인**:
- **Reflexive**: $gH = gH$ ✓
- **Symmetric**: $g_1H = g_2H \Rightarrow g_2H = g_1H$ ✓
- **Transitive**: $g_1H = g_2H, g_2H = g_3H \Rightarrow g_1H = g_3H$ ✓

**Partition**:
$$G = \bigsqcup_{i=1}^k g_iH$$

where $k = [G : H]$ (number of cosets)

### Step 3: Counting

$$|G| = \sum_{i=1}^k |g_iH| = \sum_{i=1}^k |H| = k \cdot |H| = [G : H] \cdot |H|$$

따라서 $|H| \mid |G|$ ✓

# <span class="header-examples">Examples</span>

## Example 1: $S_3$ and $A_3$

$G = S_3$ (order 6), $H = A_3$ (order 3)

**Lagrange**: $|A_3| = 3 \mid 6 = |S_3|$ ✓

**Index**: $[S_3 : A_3] = 6/3 = 2$

**Cosets**:
- $A_3 = \{e, (123), (132)\}$
- $(12)A_3 = \{(12), (12)(123), (12)(132)\} = \{(12), (23), (13)\}$

**확인**: $S_3 = A_3 \sqcup (12)A_3$ ✓

자세한 내용은 [[Symmetric Group]] 참조

## Example 2: Cyclic Groups

$G = \mathbb{Z}/12\mathbb{Z}$, $H = \langle [3] \rangle = \{[0], [3], [6], [9]\}$

**Orders**: $|H| = 4$, $|G| = 12$

**Lagrange**: $4 \mid 12$ ✓

**Index**: $[G : H] = 12/4 = 3$

**Cosets**:
- $[0] + H = \{[0], [3], [6], [9]\}$
- $[1] + H = \{[1], [4], [7], [10]\}$
- $[2] + H = \{[2], [5], [8], [11]\}$

자세한 내용은 [[Cyclic Group]] 참조

## Example 3: Klein Four-Group

$G = V_4 = \{e, a, b, c\}$ (order 4)

**Subgroups**:
- $\{e\}$ (order 1): $1 \mid 4$ ✓
- $\{e, a\}$ (order 2): $2 \mid 4$ ✓
- $\{e, b\}$ (order 2): $2 \mid 4$ ✓
- $\{e, c\}$ (order 2): $2 \mid 4$ ✓
- $V_4$ (order 4): $4 \mid 4$ ✓

**모든 subgroup orders가 $|V_4| = 4$를 나눔** ✓

## Example 4: Order of Element

$g \in G$, consider $H = \langle g \rangle$ (cyclic subgroup generated by $g$)

**Lagrange**: $|g| = |\langle g \rangle| \mid |G|$

**따름정리**: 원소의 order는 group의 order를 나눔

**예**: $S_3$에서
- $|(123)| = 3 \mid 6$ ✓
- $|(12)| = 2 \mid 6$ ✓

## Example 5: Prime Order Groups

$|G| = p$ (prime)

**Lagrange**: Subgroup $H$에 대해 $|H| \mid p$

$p$ prime이므로 $|H| = 1$ or $|H| = p$

**따라서**: $H = \{e\}$ or $H = G$ (proper nontrivial subgroup 없음)

**결론**: Prime order group은 cyclic

자세한 내용은 [[Simple Group]] 참조

# <span class="header-properties">Properties</span>

## Corollaries^[따름정리]

### 1. Order of Element Divides Group Order

$g \in G$ finite $\Rightarrow |g| \mid |G|$

**증명**: $|\langle g \rangle| = |g|$ and Lagrange's theorem ✓

### 2. $g^{|G|} = e$ for all $g \in G$

**증명**: $|g| \mid |G| \Rightarrow |G| = k|g|$ for some $k$

$$g^{|G|} = g^{k|g|} = (g^{|g|})^k = e^k = e$$

### 3. Prime Order Implies Cyclic

$|G| = p$ prime $\Rightarrow G \cong \mathbb{Z}/p\mathbb{Z}$

**증명**: 
- $g \neq e$이면 $|g| \mid p$이고 $|g| > 1$
- $p$ prime이므로 $|g| = p$
- 따라서 $\langle g \rangle = G$ (cyclic) ✓

자세한 내용은 [[Cyclic Group]] 참조

### 4. Group of Order $pq$ ($p < q$ primes)

특별한 구조:
- $q \not\equiv 1 \pmod{p}$이면 cyclic
- Sylow theorems 적용 가능

자세한 내용은 [[Subgroup]] 참조

## Converse is False^[역은 거짓]

**역**: $d \mid |G| \Rightarrow \exists H \leq G$ with $|H| = d$?

**일반적으로 거짓!**

### Counterexample: $A_4$

$|A_4| = 12 = 2^2 \cdot 3$

**$6 \mid 12$ 하지만 $A_4$에 order 6인 subgroup 없음!**

**이유**: 
- Order 6 subgroup이면 index 2
- Index 2 subgroup은 normal
- $A_4$의 normal subgroups: $\{e\}, V_4, A_4$ only
- $V_4$의 order = 4 ≠ 6 ✗

### When Converse Holds

**Abelian groups**: 역이 성립! ✓

**정리**: $G$ abelian, $d \mid |G| \Rightarrow \exists H \leq G$ with $|H| = d$

**증명**: Fundamental theorem of finitely generated abelian groups

자세한 내용은 [[Abelian Group]] 참조

## Index and Transitivity

### Product Formula

$K \leq H \leq G$

$$[G : K] = [G : H] \cdot [H : K]$$

**증명**: Cosets의 counting

**응용**: Index calculations

### Tower Law

Analogous to field theory tower law:

$$[K : F] = [K : E] \cdot [E : F]$$

## Fermat's Little Theorem Connection

**Lagrange's theorem in $(\mathbb{Z}/p\mathbb{Z})^\times$**

$|(\mathbb{Z}/p\mathbb{Z})^\times| = p-1$

$a \in (\mathbb{Z}/p\mathbb{Z})^\times \Rightarrow |a| \mid (p-1)$

$$a^{p-1} \equiv 1 \pmod{p}$$

**Fermat's Little Theorem**!

자세한 내용은 [[Euler's theorem]] 참조

# <span class="header-theorem">Theorem</span>

## Cauchy's Theorem

**정리**: $p$ prime, $p \mid |G| \Rightarrow \exists g \in G$ with $|g| = p$

**Lagrange와 관계**:
- Lagrange: $|g| \mid |G|$
- Cauchy: $p \mid |G| \Rightarrow$ order $p$인 원소 존재

**증명**: Group action + counting

## Sylow Theorems

Lagrange's theorem의 강력한 일반화

**Sylow's First Theorem**: $p^k \mid |G| \Rightarrow \exists H \leq G$ with $|H| = p^k$

**의미**: Prime power divisors에 대해서는 역이 성립!

자세한 내용은 [[Subgroup]] 참조

## Class Equation

$$|G| = |Z(G)| + \sum_i [G : C_G(x_i)]$$

**Lagrange 사용**: $[G : C_G(x_i)] = |G|/|C_G(x_i)|$

**응용**: $p$-groups, Nilpotent groups

자세한 내용은 [[Nilpotent Group]] 참조

# <span class="header-remark">Remark</span>

## 직관적 이해

**Lagrange's Theorem = "부분은 전체를 나눈다"**

### 메타포: 타일링

**Group** $G$: 바닥

**Subgroup** $H$: 타일 하나

**Cosets**: 타일들로 바닥을 완전히 덮음

**Lagrange**: 타일 개수 × 타일 크기 = 바닥 크기

### 메타포: 나눗셈

$$G = H \sqcup g_2H \sqcup \cdots \sqcup g_kH$$

$k$ = 몇 조각으로 나누나 = Index = $|G|/|H|$

## 왜 중요한가?

### 1. 기본 제약 조건

Subgroup의 가능한 orders:
- $|G| = 12$이면
- Possible subgroup orders: 1, 2, 3, 4, 6, 12 only

**응용**: Group 구조 분석

### 2. Element Order

$$|g| \mid |G|$$

**따라서**: $g^{|G|} = e$ (모든 원소)

### 3. 분류 정리 기초

Small order groups 분류:
- Order $p$ (prime): Cyclic (only one up to isomorphism)
- Order $p^2$: Abelian (two types)
- Order $pq$: 구조 제약

### 4. Number Theory 연결

**Fermat's Little Theorem**: Lagrange in $(\mathbb{Z}/p\mathbb{Z})^\times$

**Euler's Theorem**: Lagrange in $(\mathbb{Z}/n\mathbb{Z})^\times$

## 역이 성립하지 않는 이유

**Lagrange**: $|H| \mid |G|$ (항상)

**역**: $d \mid |G| \Rightarrow \exists H$ with $|H| = d$? (일반적으로 거짓)

**왜?**: 
- Subgroup은 단순히 subset이 아님
- Group 구조를 보존해야
- 특정 orders는 구조적으로 불가능할 수 있음

**예외**: Abelian groups, Sylow $p$-subgroups

## Lagrange Before Lagrange

**역사**: Joseph-Louis Lagrange (1770s)

연구 주제: Polynomial equations, Permutations

**실제 정리**: 나중에 Galois, Cauchy가 정리화

**현대 이름**: Lagrange를 기리기 위해

## 응용 사례

### Example: No Subgroup of Order 6 in $A_4$

$|A_4| = 12$, $6 \mid 12$

**하지만**: No subgroup of order 6

**이유**: 
- Order 6 $\Rightarrow$ Index 2 $\Rightarrow$ Normal
- $A_4$의 normal subgroups: $\{e\}, V_4, A_4$
- None has order 6 ✗

### Example: Simple Groups

$|G|$ prime $\Rightarrow G$ simple

**증명**: 
- Subgroups: $\{e\}, G$ only (Lagrange)
- Therefore simple ✓

자세한 내용은 [[Simple Group]] 참조

### $G = S_4$, $|G| = 24$

Possible subgroup orders: 1, 2, 3, 4, 6, 8, 12, 24

**Examples**:
- Order 1: $\{e\}$ ✓
- Order 2: $\langle (12) \rangle$ ✓
- Order 3: $\langle (123) \rangle$ ✓
- Order 4: $V_4 = \{e, (12)(34), (13)(24), (14)(23)\}$ ✓

- Order 12: $A_4$ ✓

**모든 가능한 orders가 실제로 나타남** (우연!)

Possible: 1, 2, 3, 4, 6, 12

**실제**:
- 1 ✓, 2 ✓, 3 ✓, 4 ✓, 6 ✗, 12 ✓

**6은 없음!** (Lagrange 역의 반례)

| 표기 | 의미 |
|------|------|
| $[G : H]$ | Index of $H$ in $G$ |

| $Hg$ | Right coset |

## Common Pitfall^[흔한 실수]

### 1. 역은 거짓

$d \mid |G|$ $\not\Rightarrow$ $\exists H$ with $|H| = d$ (일반적으로)

**Abelian case**: 역 성립 ✓

### 2. Left coset ≠ Subgroup

$gH$는 일반적으로 subgroup 아님 (unless $g \in H$)

### 3. $|gH| = |H|$ but $gH \neq H$ (일반적으로)

Coset는 같은 크기지만 다른 집합

### 4. Index는 나눗셈

$[G : H] = |G|/|H|$ (finite case)

크기의 비율 = Cosets 개수

### 5. $g^{|G|} = e$는 minimal이 아님

$|g|$가 minimal exponent, not $|G|$
