# Footnote 테스트

이 파일은 Obsidian에서 footnote가 어떻게 보이는지 테스트하기 위한 파일입니다.

---

## 사용 예시

### 기본 예시
마우스를 아래 단어 위에 올려보세요:

linear mapping^[선형 사상]은 vector space^[벡터 공간] 사이의 구조를 보존하는 함수입니다.

### 수학적 정의
$T: V \rightarrow W$가 linear mapping^[선형 사상]이면, kernel^[핵] $\ker(T)$와 image^[상] $\text{Im}(T)$를 정의할 수 있습니다.

### 성질
- $T$가 injective^[단사]이다 $\Leftrightarrow$ $\ker(T) = \{0\}$
- $T$가 surjective^[전사]이다 $\Leftrightarrow$ $\text{Im}(T) = W$
- $T$가 isomorphism^[동형사상]이다 $\Leftrightarrow$ $T$가 injective이면서 surjective

### 더 복잡한 예시
Hilbert space에서 bounded linear functional^[유계 선형 범함수]은 inner product^[내적]로 표현할 수 있습니다. 이것이 Riesz Representation Theorem입니다.

---

## Footnote의 장점

1. **깔끔한 표시**: 괄호 없이 깔끔하게 보임
2. **선택적 확인**: 필요할 때만 마우스를 올려서 확인
3. **읽기 모드**: 읽기 모드에서도 자연스럽게 작동
