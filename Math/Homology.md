# <span class="header-definition">Definition</span>

## Homology^[호모로지]

**Homology^[호모로지]**: 위상공간 \(X\)에 **abelian groups^[아벨군]** \(H_n(X)\)를 대응시키는 functor^[함자]

$$H_n: \mathbf{Top} \to \mathbf{Ab}, \quad X \mapsto H_n(X)$$

**핵심 아이디어**: "위상공간의 \(n\)차원 holes^[구멍]를 측정"

**의미**:
- \(H_0(X)\): Connected components^[연결 성분]
- \(H_1(X)\): Loops^[고리] (1차원 구멍)
- \(H_2(X)\): Voids^[공동] (2차원 구멍)
- \(H_n(X)\): \(n\)-dimensional holes

## Singular Homology^[특이 호모로지]

**가장 일반적인 homology 이론**

### Singular Simplex^[특이 단체]

**Standard \(n\)-simplex^[표준 n-단체]**:

$$\Delta^n = \{(t_0, \ldots, t_n) \in \mathbb{R}^{n+1} : t_i \geq 0, \sum_{i=0}^n t_i = 1\}$$

**예**:
- \(\Delta^0\): Point
- \(\Delta^1\): Line segment \([0, 1]\)
- \(\Delta^2\): Triangle (2-simplex)
- \(\Delta^3\): Tetrahedron (3-simplex)

**Singular \(n\)-simplex^[특이 n-단체]** on \(X\):

$$\sigma: \Delta^n \to X \quad \text{(continuous map)}$$

### Singular Chain Group^[특이 사슬군]

$$C_n(X) = \text{Free abelian group generated by singular } n\text{-simplices}$$

**원소**: Formal linear combinations^[형식적 일차결합]

$$c = \sum_{i} a_i \sigma_i, \quad a_i \in \mathbb{Z}, \sigma_i: \Delta^n \to X$$

### Boundary Operator^[경계 작용소]

**Face maps^[면 사상]**: \(i\)-th face of \(\Delta^n\)

$$d_i: \Delta^{n-1} \to \Delta^n, \quad d_i(t_0, \ldots, t_{n-1}) = (t_0, \ldots, t_{i-1}, 0, t_i, \ldots, t_{n-1})$$

**Boundary operator^[경계 작용소]**: \(\partial_n: C_n(X) \to C_{n-1}(X)\)

$$\partial_n(\sigma) = \sum_{i=0}^n (-1)^i \sigma \circ d_i$$

**핵심 성질**: \(\partial_{n-1} \circ \partial_n = 0\)

$$\partial^2 = 0$$

### Chain Complex^[사슬 복체]

$$\cdots \xrightarrow{\partial_{n+2}} C_{n+1}(X) \xrightarrow{\partial_{n+1}} C_n(X) \xrightarrow{\partial_n} C_{n-1}(X) \xrightarrow{\partial_{n-1}} \cdots \xrightarrow{\partial_1} C_0(X) \xrightarrow{\partial_0} 0$$

자세한 내용은 [[Chain Complex]] 참조

### Cycle and Boundary^[순환과 경계]

**\(n\)-Cycle^[n-순환]**: 

$$Z_n(X) = \ker(\partial_n) = \{c \in C_n(X) : \partial_n(c) = 0\}$$

"경계가 없는 chain"

**\(n\)-Boundary^[n-경계]**:

$$B_n(X) = \text{im}(\partial_{n+1}) = \{\partial_{n+1}(c) : c \in C_{n+1}(X)\}$$

"어떤 chain의 경계"

**중요**: \(\partial^2 = 0\) \(\Rightarrow\) \(B_n(X) \subseteq Z_n(X)\)

### Homology Group^[호모로지군]

$$H_n(X) = Z_n(X) / B_n(X) = \ker(\partial_n) / \text{im}(\partial_{n+1})$$

**의미**: "경계가 아닌 순환" / "trivial한 순환"

**직관**: Cycles modulo boundaries

## Reduced Homology^[축약 호모로지]

**정의**: Augmented chain complex^[증대 사슬 복체]

$$\cdots \to C_1(X) \xrightarrow{\partial_1} C_0(X) \xrightarrow{\varepsilon} \mathbb{Z} \to 0$$

where \(\varepsilon\): Augmentation map^[증대 사상]

$$\varepsilon\left(\sum_i a_i \sigma_i\right) = \sum_i a_i$$

**Reduced homology^[축약 호모로지]**:

$$\tilde{H}_n(X) = \begin{cases} \ker(\varepsilon) / \text{im}(\partial_1) & n = 0 \\ H_n(X) & n > 0 \end{cases}$$

**관계**:

$$H_0(X) \cong \tilde{H}_0(X) \oplus \mathbb{Z}$$

**장점**: \(\tilde{H}_0(\text{pt}) = 0\) (단일점의 0차 호모로지가 0)

## Relative Homology^[상대 호모로지]

**Setup**: Pair^[쌍] \((X, A)\) where \(A \subseteq X\)

**Relative chain group^[상대 사슬군]**:

$$C_n(X, A) = C_n(X) / C_n(A)$$

**Relative homology^[상대 호모로지]**:

$$H_n(X, A) = H_n(C_*(X, A))$$

**의미**: "\(X\)의 homology relative to \(A\)"

**Long exact sequence^[긴 완전 수열]**:

$$\cdots \to H_n(A) \to H_n(X) \to H_n(X, A) \to H_{n-1}(A) \to \cdots$$

자세한 내용은 [[Exact Sequence]] 참조

## Simplicial Homology^[단체 호모로지]

**Setup**: Simplicial complex^[단체 복체] \(K\)

**\(n\)-Simplex^[n-단체]**: \([v_0, v_1, \ldots, v_n]\) (vertices \(v_i\))

**Simplicial chain group^[단체 사슬군]**:

$$C_n(K) = \text{Free abelian group on } n\text{-simplices of } K$$

**Boundary operator^[경계 작용소]**:

$$\partial_n([v_0, \ldots, v_n]) = \sum_{i=0}^n (-1)^i [v_0, \ldots, \hat{v}_i, \ldots, v_n]$$

(where \(\hat{v}_i\) means omit \(v_i\))

**Simplicial homology^[단체 호모로지]**:

$$H_n(K) = \ker(\partial_n) / \text{im}(\partial_{n+1})$$

**관계**: For triangulable spaces^[삼각분할 가능 공간], simplicial = singular homology

## Cellular Homology^[셀룰러 호모로지]

**Setup**: CW complex^[CW 복체] \(X\)

**Cellular chain group^[셀룰러 사슬군]**:

$$C_n^{CW}(X) = H_n(X^n, X^{n-1})$$

where \(X^n\): \(n\)-skeleton^[n-골격]

**Cellular boundary map^[셀룰러 경계 사상]**: Degree of attaching map

**정리**: \(H_n^{CW}(X) \cong H_n(X)\) (singular homology)

**장점**: 계산이 훨씬 간단!

자세한 내용은 [[CW Complex]] 참조

# <span class="header-properties">Properties</span>

## Functoriality^[함자성]

**Continuous map^[연속 사상]** \(f: X \to Y\)는 homomorphism^[준동형사상]을 유도:

$$f_*: H_n(X) \to H_n(Y)$$

**성질**:
1. \((\text{id}_X)_* = \text{id}_{H_n(X)}\)
2. \((g \circ f)_* = g_* \circ f_*\)

**의미**: \(H_n\)은 functor^[함자]

자세한 내용은 [[Category Theory]] 참조

## Homotopy Invariance^[호모토피 불변성]

**정리**: \(f \simeq g: X \to Y\) (homotopic^[호모토피])이면:

$$f_* = g_*: H_n(X) \to H_n(Y)$$

**따름정리**: Homotopy equivalent^[호모토피 동치] spaces have isomorphic homology

$$X \simeq Y \Rightarrow H_n(X) \cong H_n(Y)$$

**예**: Contractible space^[축약가능 공간] \(X \simeq \text{pt}\)이면:

$$H_n(X) = \begin{cases} \mathbb{Z} & n = 0 \\ 0 & n > 0 \end{cases}$$

## Dimension Axiom^[차원 공리]

**정리**: For single point \(\text{pt}\):

$$H_n(\text{pt}) = \begin{cases} \mathbb{Z} & n = 0 \\ 0 & n > 0 \end{cases}$$

**Reduced version**:

$$\tilde{H}_n(\text{pt}) = 0 \quad \forall n$$

## Exactness^[완전성]

**Long exact sequence of pair^[쌍의 긴 완전 수열]**:

$$\cdots \to H_n(A) \to H_n(X) \to H_n(X, A) \to H_{n-1}(A) \to \cdots$$

자세한 내용은 [[Exact Sequence]] 참조

## Excision Theorem^[절제 정리]

**정리**: \(X = A \cup B\), \(\bar{Z} \subseteq \text{int}(B)\)이면:

$$H_n(X \setminus Z, A \setminus Z) \cong H_n(X, A)$$

**의미**: "Removing \(Z\) doesn't change relative homology"

**응용**: Mayer-Vietoris sequence 증명

자세한 내용은 [[Mayer–Vietoris sequence]] 참조

## Additivity^[가법성]

**정리**: Disjoint union^[분리합집합]

$$H_n\left(\bigsqcup_\alpha X_\alpha\right) \cong \bigoplus_\alpha H_n(X_\alpha)$$

**예**: 

$$H_n(X \sqcup Y) \cong H_n(X) \oplus H_n(Y)$$

## Mayer-Vietoris Sequence^[마이어-비에토리스 수열]

**정리**: \(X = A \cup B\) (open)이면:

$$\cdots \to H_n(A \cap B) \to H_n(A) \oplus H_n(B) \to H_n(X) \to H_{n-1}(A \cap B) \to \cdots$$

**가장 강력한 계산 도구**

자세한 내용은 [[Mayer–Vietoris sequence]] 참조

## Hurewicz Theorem^[후레비츠 정리]

**정리**: Simply connected^[단일 연결] space \(X\)에 대해:

$$H_1(X) = 0$$

일반적으로:

$$\pi_1(X)^{\text{ab}} \cong H_1(X)$$

where \(\pi_1(X)^{\text{ab}}\): Abelianization of fundamental group

자세한 내용은 [[Fundamental Group]] 참조

## Universal Coefficient Theorem^[보편 계수 정리]

**정리**: For any abelian group \(G\):

$$0 \to H_n(X) \otimes G \to H_n(X; G) \to \text{Tor}(H_{n-1}(X), G) \to 0$$

**Split exact sequence**

**의미**: Homology with coefficients^[계수를 갖는 호모로지] determined by homology with \(\mathbb{Z}\)

## Künneth Formula^[퀴네트 공식]

**정리**: For product spaces:

$$H_n(X \times Y) \cong \bigoplus_{p+q=n} (H_p(X) \otimes H_q(Y)) \oplus \bigoplus_{p+q=n-1} \text{Tor}(H_p(X), H_q(Y))$$

**Tor-free case** (e.g., \(H_*(X)\) free):

$$H_n(X \times Y) \cong \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y)$$

# <span class="header-examples">Examples</span>

## Example 1: Point

$$H_n(\text{pt}) = \begin{cases} \mathbb{Z} & n = 0 \\ 0 & n > 0 \end{cases}$$

**의미**: 0개의 holes (except 1 component)

## Example 2: Circle \(S^1\)

$$H_n(S^1) = \begin{cases} \mathbb{Z} & n = 0, 1 \\ 0 & n \geq 2 \end{cases}$$

**의미**:
- \(H_0 = \mathbb{Z}\): 1 component
- \(H_1 = \mathbb{Z}\): 1 loop (hole)
- \(H_n = 0\) (\(n \geq 2\)): No higher holes

## Example 3: Sphere \(S^n\)

$$H_k(S^n) = \begin{cases} \mathbb{Z} & k = 0, n \\ 0 & \text{otherwise} \end{cases}$$

**의미**: \(n\)-sphere has one \(n\)-dimensional hole

**계산**: Mayer-Vietoris (두 hemispheres)

## Example 4: Torus \(T^2 = S^1 \times S^1\)

$$H_n(T^2) = \begin{cases} \mathbb{Z} & n = 0, 2 \\ \mathbb{Z}^2 & n = 1 \\ 0 & n \geq 3 \end{cases}$$

**의미**:
- \(H_0 = \mathbb{Z}\): 1 component
- \(H_1 = \mathbb{Z}^2\): 2 independent loops
- \(H_2 = \mathbb{Z}\): 1 void (interior)

**계산**: Künneth formula or Mayer-Vietoris

## Example 5: Real Projective Space \(\mathbb{RP}^n\)

$$H_k(\mathbb{RP}^n) = \begin{cases} 
\mathbb{Z} & k = 0 \\
\mathbb{Z}/2\mathbb{Z} & k \text{ odd}, 0 < k < n \\
\mathbb{Z} & k = n \text{ (if } n \text{ odd)} \\
0 & \text{otherwise}
\end{cases}$$

**예** (\(\mathbb{RP}^2\)):

$$H_*(\mathbb{RP}^2) = (\mathbb{Z}, \mathbb{Z}/2\mathbb{Z}, 0, 0, \ldots)$$

## Example 6: Complex Projective Space \(\mathbb{CP}^n\)

$$H_k(\mathbb{CP}^n) = \begin{cases} 
\mathbb{Z} & k = 0, 2, 4, \ldots, 2n \\
0 & \text{otherwise}
\end{cases}$$

**예** (\(\mathbb{CP}^2\)):

$$H_*(\mathbb{CP}^2) = (\mathbb{Z}, 0, \mathbb{Z}, 0, \mathbb{Z}, 0, \ldots)$$

## Example 7: Klein Bottle \(K\)

$$H_n(K) = \begin{cases} 
\mathbb{Z} & n = 0 \\
\mathbb{Z} \oplus \mathbb{Z}/2\mathbb{Z} & n = 1 \\
0 & n \geq 2
\end{cases}$$

**의미**: Non-orientable^[방향불가능] (torsion in \(H_1\))

## Example 8: Figure Eight \(S^1 \vee S^1\)

$$H_n(S^1 \vee S^1) = \begin{cases} 
\mathbb{Z} & n = 0 \\
\mathbb{Z}^2 & n = 1 \\
0 & n \geq 2
\end{cases}$$

**의미**: 2 independent loops

## Example 9: \(n\)-Torus \(T^n\)

$$H_k(T^n) \cong \mathbb{Z}^{\binom{n}{k}}$$

**예** (\(T^3\)):

$$H_*(T^3) = (\mathbb{Z}, \mathbb{Z}^3, \mathbb{Z}^3, \mathbb{Z}, 0, \ldots)$$

**계산**: Künneth formula

## Example 10: Disk \(D^n\)

Contractible^[축약가능]이므로:

$$H_k(D^n) = \begin{cases} 
\mathbb{Z} & k = 0 \\
0 & k > 0
\end{cases}$$

# <span class="header-theorem">Theorem</span>

## Eilenberg-Steenrod Axioms^[아일렌베르그-스틴로드 공리]

**정리**: Homology는 다음 axioms를 만족:

1. **Homotopy**: \(f \simeq g \Rightarrow f_* = g_*\)
2. **Exactness**: Long exact sequence of pair
3. **Excision**: Excision theorem
4. **Dimension**: \(H_n(\text{pt})\) as above
5. **Additivity**: Disjoint unions

**유일성**: 이 axioms가 homology theory를 uniquely determine

## Hurewicz Theorem^[후레비츠 정리]

**정리**: Path-connected^[경로 연결] \(X\)에 대해:

$$h: \pi_n(X) \to H_n(X)$$ 

**Hurewicz homomorphism^[후레비츠 준동형사상]**

**Case 1**: \(n = 1\)

$$h: \pi_1(X) \to H_1(X)$$

Surjective, kernel = commutator subgroup^[교환자 부분군]

$$H_1(X) \cong \pi_1(X) / [\pi_1(X), \pi_1(X)]$$

**Case 2**: \(X\) simply connected, \(\pi_i(X) = 0\) for \(i < n\)

$$h: \pi_n(X) \xrightarrow{\cong} H_n(X)$$

Isomorphism!

자세한 내용은 [[Homotopy Group]] 참조

## Poincaré Duality^[푸앵카레 쌍대성]

**정리**: Closed^[닫힌] orientable^[방향가능] \(n\)-manifold \(M\)에 대해:

$$H_k(M) \cong H^{n-k}(M)$$

**의미**: Homology ↔ Cohomology (dimension complementary)

**예** (Torus \(T^2\)):

$$H_0(T^2) \cong H^2(T^2) \cong \mathbb{Z}$$

$$H_1(T^2) \cong H^1(T^2) \cong \mathbb{Z}^2$$

자세한 내용은 [[Cohomology]] 참조

## Lefschetz Fixed Point Theorem^[레프셰츠 고정점 정리]

**Lefschetz number^[레프셰츠 수]**:

$$L(f) = \sum_{n=0}^\infty (-1)^n \text{tr}(f_*: H_n(X) \to H_n(X))$$

**정리**: \(L(f) \neq 0\) \(\Rightarrow\) \(f\) has fixed point

**응용**: Prove existence of fixed points

자세한 내용은 [[Fixed Point Theorem]] 참조

## Betti Numbers^[베티 수]

**정의**: \(n\)-th Betti number^[n차 베티 수]

$$b_n(X) = \text{rank}(H_n(X))$$

(rank of free part)

**Euler characteristic^[오일러 지표]**:

$$\chi(X) = \sum_{n=0}^\infty (-1)^n b_n(X)$$

**예** (Torus): \(b_0 = 1, b_1 = 2, b_2 = 1\)

$$\chi(T^2) = 1 - 2 + 1 = 0$$

자세한 내용은 [[Euler Characteristic]] 참조

## Universal Coefficient Theorem (Detailed)^[보편 계수 정리]

**For homology**:

$$0 \to H_n(X) \otimes G \to H_n(X; G) \to \text{Tor}(H_{n-1}(X), G) \to 0$$

**For cohomology**:

$$0 \to \text{Ext}(H_{n-1}(X), G) \to H^n(X; G) \to \text{Hom}(H_n(X), G) \to 0$$

**의미**: Coefficients in any abelian group \(G\)

## Künneth Theorem (Detailed)^[퀴네트 정리]

**정리**: Short exact sequence

$$0 \to \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y) \to H_n(X \times Y) \to \bigoplus_{p+q=n-1} \text{Tor}(H_p(X), H_q(Y)) \to 0$$

**Tor-free case**:

$$H_n(X \times Y) \cong \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y)$$

# <span class="header-proof">Proof</span>

## \(\partial^2 = 0\) (Key Property)

**정리**: \(\partial_{n-1} \circ \partial_n = 0\)

**증명**: For singular chain

$$\partial^2(\sigma) = \partial\left(\sum_{i=0}^n (-1)^i \sigma \circ d_i\right)$$

$$= \sum_{i=0}^n (-1)^i \partial(\sigma \circ d_i)$$

$$= \sum_{i=0}^n \sum_{j=0}^{n-1} (-1)^{i+j} \sigma \circ d_i \circ d_j$$

**Key observation**: Each term appears twice with opposite signs:

- \(\sigma \circ d_i \circ d_j\) (for \(j < i\)): coefficient \((-1)^{i+j}\)
- \(\sigma \circ d_{i+1} \circ d_j\) (for \(j \geq i\)): coefficient \((-1)^{i+1+j}\)

**Cancellation**: All terms cancel \(\Rightarrow\) \(\partial^2 = 0\)

## Homotopy Invariance Proof Sketch

**정리**: \(f \simeq g\) \(\Rightarrow\) \(f_* = g_*\)

**증명 idea**: Construct **chain homotopy^[사슬 호모토피]** \(P: C_n(X) \to C_{n+1}(Y)\)

$$\partial P + P \partial = g_\# - f_\#$$

**따름정리**: \(g_* - f_* = 0\) in homology

자세한 내용은 [[Homotopy]] 참조

## Homology of \(S^n\) (via Mayer-Vietoris)

**Setup**: \(S^n = U \cup V\) (upper/lower hemispheres)

- \(U, V \simeq \text{pt}\) (contractible)
- \(U \cap V \simeq S^{n-1}\)

**Mayer-Vietoris**:

$$H_k(S^{n-1}) \to H_k(U) \oplus H_k(V) \to H_k(S^n) \to H_{k-1}(S^{n-1})$$

**For \(0 < k < n\)**:

$$H_k(S^{n-1}) \to 0 \to H_k(S^n) \to H_{k-1}(S^{n-1})$$

**Induction**: \(H_k(S^n) \cong H_{k-1}(S^{n-1})\) for \(0 < k < n\)

**Base**: \(H_0(S^0) = \mathbb{Z}^2\), \(H_k(S^0) = 0\) (\(k > 0\))

**결과**:

$$H_k(S^n) = \begin{cases} \mathbb{Z} & k = 0, n \\ 0 & \text{otherwise} \end{cases}$$

# <span class="header-remark">Remark</span>

## 직관적 이해

**핵심 아이디어**: "Counting holes^[구멍 세기]"

**메타포**:
- \(H_0\): Connected components (separate pieces)
- \(H_1\): Loops (circular holes)
- \(H_2\): Voids (hollow regions)
- \(H_n\): \(n\)-dimensional holes

**예**: Coffee cup
- \(H_0 = \mathbb{Z}\): 1 piece
- \(H_1 = \mathbb{Z}\): 1 handle (loop)
- \(H_2 = 0\): No void (solid cup)

## Cycle vs Boundary

**Cycle^[순환]**: "Closed" chain (\(\partial c = 0\))

**Boundary^[경계]**: "Bounding" chain (\(c = \partial d\))

**Homology**: Cycles that are NOT boundaries

**직관**: 
- Cycle: Loop around a hole
- Boundary: Loop that bounds a disk
- Homology: "Essential" loops (non-trivial holes)

## Homology vs Homotopy

| Concept | Homology \(H_n\) | Homotopy \(\pi_n\) |
|---------|-----------------|-------------------|
| **Type** | Abelian group | Group (non-abelian for \(n = 1\)) |
| **Computation** | Easier (linear algebra) | Harder (non-linear) |
| **Detects** | Holes | Loops/Spheres |
| **\(S^2\)** | \(H_2(S^2) = \mathbb{Z}\) | \(\pi_2(S^2) = \mathbb{Z}\) |
| **\(S^3\)** | \(H_3(S^3) = \mathbb{Z}\) | \(\pi_3(S^3) = \mathbb{Z}\) |
| **Higher** | \(H_n(S^n) = \mathbb{Z}\) | \(\pi_k(S^n)\) very complex! |

**관계**: Hurewicz theorem connects them

자세한 내용은 [[Homotopy Group]] 참조

- Homology as groups (not just numbers)
- Abstract algebra approach

- Mayer-Vietoris sequence

- Axiomatization of homology

## Comparison: Homology Theories

| Theory | Input | Computation | Generality |
|--------|-------|-------------|------------|
| **Singular** | Any space | Hard | Universal |
| **Simplicial** | Simplicial complex | Matrix algebra | Combinatorial |
| **Cellular** | CW complex | Easier | Very useful |
| **de Rham** | Smooth manifold | Differential forms | Smooth only |

## Common Pitfalls^[흔한 실수]

### 1. Cycle ≠ Loop

✗ "Cycle = loop in space"?

✓ **Cycle = formal sum of simplices with \(\partial = 0\)**

### 2. Homology ≠ Homotopy

✗ "\(H_n(X) = \pi_n(X)\)"?

✓ **Different!** Hurewicz connects them

**예**: \(\pi_3(S^2) = \mathbb{Z}\) but \(H_3(S^2) = 0\)

### 3. Torsion

✗ "Homology always free abelian"?

✓ **Can have torsion!** (e.g., \(H_1(K) = \mathbb{Z} \oplus \mathbb{Z}/2\mathbb{Z}\))

### 4. Reduced vs Unreduced

✗ "\(H_0(\text{pt}) = 0\)"?

✓ **Unreduced**: \(H_0(\text{pt}) = \mathbb{Z}\)

✓ **Reduced**: \(\tilde{H}_0(\text{pt}) = 0\)

### 5. Relative Homology

✗ "\(H_n(X, A) = H_n(X) / H_n(A)\)"?

✓ **Not always!** Only in special cases

## Computational Strategy^[계산 전략]

**How to compute \(H_*(X)\)**:

### Method 1: Mayer-Vietoris

1. Decompose \(X = A \cup B\)
2. Know \(H_*(A), H_*(B), H_*(A \cap B)\)
3. Use long exact sequence

### Method 2: CW Complex

1. Give \(X\) a CW structure
2. Compute cellular chains
3. Matrix algebra (boundary maps)

### Method 3: Known Results

1. Check if \(X \simeq Y\) (homotopy equivalent)
2. Use \(H_*(X) \cong H_*(Y)\)

### Method 4: Long Exact Sequence

1. Find pair \((X, A)\) or fibration
2. Use exactness

### Method 5: Künneth

1. If \(X = Y \times Z\)
2. Use Künneth formula

**예**: \(H_*(T^2)\)
- Method 1: Mayer-Vietoris (two solid tori)
- Method 2: CW complex (1 vertex, 2 edges, 1 face)
- Method 3: \(T^2 = S^1 \times S^1\) (Künneth)
