# <span class="header-definition">Definition</span>

## Commutator in Ring^[환에서의 교환자]

Ring $R$의 원소 $a, b \in R$에 대한 **commutator**^[교환자]:

$$[a, b] = ab - ba$$

**의미**: $a$와 $b$가 교환하지 않는 정도를 측정

### 비교: Group의 Commutator

**Group**: $[g, h] = ghg^{-1}h^{-1}$

**Ring**: $[a, b] = ab - ba$

**차이**: Ring에서는 역원 개념 없이 뺄셈 사용

자세한 내용은 [[Commutator]] 참조

## Anticommutator^[반교환자]

Ring $R$의 원소 $a, b \in R$에 대한 **anticommutator**^[반교환자]:

$$\{a, b\} = ab + ba$$

**응용**: Quantum mechanics, Jordan algebras

## Lie Bracket^[리 브라켓]

**Lie bracket**^[리 브라켓]은 commutator와 같은 표기:

$$[a, b] = ab - ba$$

**의미**: Lie algebra 구조 정의

**중요**: Associative ring에서 자연스럽게 Lie algebra 구조 유도

## Commutative Elements^[가환 원소]

$$[a, b] = 0 \Leftrightarrow ab = ba$$

**의미**: Commutator가 0 $\Leftrightarrow$ 원소들이 commute

# <span class="header-properties">Properties</span>

## Basic Properties

### 1. Anticommutativity^[반교환성]

$$[a, b] = -[b, a]$$

**증명**: $[b, a] = ba - ab = -(ab - ba) = -[a, b]$ ✓

### 2. Bilinearity^[쌍선형성]

$$[a + b, c] = [a, c] + [b, c]$$

$$[a, b + c] = [a, b] + [a, c]$$

**증명**: 
$[a+b, c] = (a+b)c - c(a+b) = ac + bc - ca - cb = [a,c] + [b,c]$ ✓

### 3. Self-commutator is Zero^[자기 자신과의 교환자는 0]

$$[a, a] = 0$$

**증명**: $[a, a] = aa - aa = 0$ ✓

### 4. Jacobi Identity^[야코비 항등식]

$$[a, [b, c]] + [b, [c, a]] + [c, [a, b]] = 0$$

**증명**: 직접 계산으로 확인 가능 (associativity 사용)

**의미**: Lie algebra의 핵심 성질!

## Derivation Property^[미분 성질]

### Leibniz Rule

$$[a, bc] = [a, b]c + b[a, c]$$

**증명**: 
$$[a, bc] = a(bc) - (bc)a = abc - bca + bca - bca$$
$$= abc - bca = (ab - ba)c + b(ac - ca) = [a,b]c + b[a,c]$$ ✓

**의미**: Commutator with $a$는 derivation처럼 작동

## Center and Commutators^[중심과 교환자]

**정리**: $a \in Z(R)$ (center) $\Leftrightarrow$ $[a, b] = 0$ for all $b \in R$

**증명**: 
$$a \in Z(R) \Leftrightarrow ab = ba \text{ for all } b \Leftrightarrow [a,b] = 0 \text{ for all } b$$ ✓

자세한 내용은 [[Center]] 참조

## Commutator Ideal^[교환자 아이디얼]

**정의**: Ring $R$의 **commutator ideal**^[교환자 아이디얼]:

$$[R, R] = \text{ideal generated by all } [a, b], \; a, b \in R$$

**성질**: $R/[R,R]$은 commutative ring (largest commutative quotient)

**비교**: Group의 commutator subgroup과 유사

자세한 내용은 [[Commutator Subgroup]] 참조

# <span class="header-examples">Examples</span>

## Example 1: Commutative Rings

**$R$ commutative** (e.g., $\mathbb{Z}$, $\mathbb{R}$, $\mathbb{Q}[x]$):

$$[a, b] = ab - ba = 0 \quad \text{for all } a, b$$

**모든 commutator = 0**

## Example 2: Matrix Ring

**$M_n(\mathbb{R})$**: $n \times n$ matrices

$$A = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}, \quad B = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$$

$$[A, B] = AB - BA = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} - \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$$

**Non-zero commutator!**

## Example 3: Quaternions

**$\mathbb{H}$**: Quaternions $\{a + bi + cj + dk\}$

$$[i, j] = ij - ji = k - (-k) = 2k$$

$$[j, k] = jk - kj = i - (-i) = 2i$$

$$[k, i] = ki - ik = j - (-j) = 2j$$

자세한 내용은 [[Ring]] 참조 (Quaternions example)

## Example 4: Differential Operators

**Ring of differential operators**:

$$D = \frac{d}{dx}, \quad M_x = \text{multiplication by } x$$

$$[D, M_x] = D \circ M_x - M_x \circ D$$

작용: $[D, M_x](f) = D(xf) - xD(f) = f + xf' - xf' = f$

따라서 $[D, M_x] = I$ (identity operator)

**응용**: Quantum mechanics (canonical commutation relations)

## Example 5: Lie Algebra $\mathfrak{gl}_n$

**$\mathfrak{gl}_n(\mathbb{R})$**: $n \times n$ matrices with Lie bracket

$$[A, B] = AB - BA$$

**Lie algebra structure**: Satisfies Jacobi identity

**Subalgebras**: $\mathfrak{sl}_n$ (traceless), $\mathfrak{so}_n$ (skew-symmetric), etc.

## Example 6: Polynomial Ring

**$R[x]$**: Polynomials (commutative)

$$[f(x), g(x)] = 0 \quad \text{for all } f, g$$

**But non-commutative polynomial rings** exist (e.g., free algebra)

# <span class="header-theorem">Theorem</span>

## Jacobi Identity

**정리**: Ring $R$에서

$$[[a, b], c] + [[b, c], a] + [[c, a], b] = 0$$

**증명**: Associativity를 사용한 직접 계산

**의미**: $(R, [\cdot, \cdot])$는 Lie algebra

## Derivation Property

**정리**: Fixed $a \in R$에 대해, map $\text{ad}_a: R \to R$

$$\text{ad}_a(b) = [a, b]$$

is a **derivation**^[미분]:

$$\text{ad}_a(bc) = \text{ad}_a(b) \cdot c + b \cdot \text{ad}_a(c)$$

**증명**: Leibniz rule 참조 ✓

## Universal Enveloping Algebra

**정리**: Every Lie algebra embeds into associative algebra

Via commutator: $[a, b] = ab - ba$ in universal enveloping algebra

**응용**: Representation theory

## Poincaré-Birkhoff-Witt Theorem

**정리**: Universal enveloping algebra of Lie algebra has specific basis

**응용**: Structure of Lie algebras, representation theory

# <span class="header-remark">Remark</span>

## 직관적 이해

**Commutator = "Non-commutativity 측정"**

### 메타포: 순서의 중요성

**Commutative**: $ab = ba$ (순서 무관)

**Non-commutative**: $ab \neq ba$ (순서 중요)

**Commutator**: $[a,b] = ab - ba$ (얼마나 다른지)

**예**: 
- 행렬 곱셈: 순서 중요!
- 회전 변환: 순서 중요!
- Commutator: 차이를 정량화

### 메타포: 물리학

**Quantum mechanics**: Position $x$ and momentum $p$

$$[x, p] = xp - px = i\hbar$$

**의미**: 동시에 정확히 측정 불가 (Heisenberg uncertainty principle)

## 왜 중요한가?

### 1. Lie Algebras

**Structure**: Commutator defines Lie algebra from associative algebra

**Classification**: Simple Lie algebras classified

자세한 내용은 [[Lie Algebra]] (if exists)

### 2. Quantum Mechanics

**Canonical commutation relations**: $[x_i, p_j] = i\hbar \delta_{ij}$

**Observables**: Self-adjoint operators

**Uncertainty principle**: From commutators

### 3. Differential Geometry

**Vector fields**: Lie bracket = commutator of vector fields

**Curvature**: Measured by commutators

### 4. Representation Theory

**Adjoint representation**: $\text{ad}_a(b) = [a,b]$

**Structure constants**: Lie algebra structure

### 5. Non-commutative Geometry

**$C^*$-algebras**: Commutators fundamental

**Quantum groups**: Non-commutative structures

## Group vs Ring Commutator

| | **Group** | **Ring** |
|---|---|---|
| **정의** | $[g,h] = ghg^{-1}h^{-1}$ | $[a,b] = ab - ba$ |
| **Inverse needed** | Yes | No |
| **Subtraction** | No | Yes |
| **Properties** | Not necessarily bilinear | Bilinear |
| **Jacobi** | Complex form | Simple form |

**공통점**: Both measure "failure to commute"

## Commutator vs Anticommutator

| | **Commutator** | **Anticommutator** |
|---|---|---|
| **정의** | $[a,b] = ab - ba$ | $\{a,b\} = ab + ba$ |
| **Sign** | Antisymmetric | Symmetric |
| **Jacobi** | Yes | No |
| **응용** | Lie algebras | Jordan algebras |

**물리**: 
- Fermions: Anticommutators
- Bosons: Commutators

## 계산 방법

### Commutator 계산

**Step 1**: Compute $ab$

**Step 2**: Compute $ba$

**Step 3**: Subtract: $[a,b] = ab - ba$

### Jacobi Identity 검증

**Step 1**: Compute $[[a,b], c]$, $[[b,c], a]$, $[[c,a], b]$

**Step 2**: Add them

**Step 3**: Verify sum = 0

### Derivation Property 검증

For $\text{ad}_a(bc) = [a,b]c + b[a,c]$:

**Expand both sides** and verify equality

## 표기법

| 표기 | 의미 |
|------|------|
| $[a, b]$ | Commutator $ab - ba$ |
| $\{a, b\}$ | Anticommutator $ab + ba$ |
| $\text{ad}_a(b)$ | Adjoint action $[a,b]$ |
| $[R, R]$ | Commutator ideal |
| $Z(R)$ | Center of ring |

## Common Pitfall^[흔한 실수]

### 1. Ring vs Group Commutator

**Different definitions!**

Group: $ghg^{-1}h^{-1}$ (requires inverses)

Ring: $ab - ba$ (just subtraction)

### 2. Commutator NOT always in center

$[a,b]$ can be non-central even if $a,b$ central-ish

### 3. Jacobi Identity

Must verify explicitly for non-associative algebras!

Automatic for associative rings ✓

### 4. Anticommutator is different

$\{a,b\} \neq [a,b]$

Different symmetries, different applications

### 5. Bilinearity uses distributivity

Requires ring structure (not just group)
